Projektuppgift: Djurvisualisering
💡 Beskrivning

Skapa en webbplats som visualiserar förhållandet mellan olika observationer av en eller flera djurarter i något län i Sverige. Det kan till exempel vara visualisering av älg- och vargobservationer eller rovfåglar och kaniner.

📌 Obligatoriska Krav

🟢 Supabase med två eller flera tabeller

🟢 PK/FK med de nya tabellerna

🟡 SQL-satser testade på Servern

🟡 Webbdesign enligt designprinciperna

🔴 Webbsidan innehåller länkar till beskrivningar av vald art

🟢 Data API så som Unsplash, Wikipedia mm.

🔴 Skicka informationen om vädret under observationsdatumet

🔴 Design av sidan med semantiska taggar

🟢 CSS-filen med alla element stylade (minst en klass och ett ID)

🟢 Leaflet med markörer som representerar djurarten

🚀 Extra Utmaningar (Icke obligatoriska)

🔴 Använd anropet till GBIF via dess API och sök direkt via ditt formulär

🟢 Manipulera kartbeteende med fler funktioner som vi inte gått igenom

🔴 Möjliggör ifyllningen av en ny kommun i formuläret
