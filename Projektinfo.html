Projekt i kursen HTML, CSS och Javascript

Projekt i kursen HTML, CSS och Javascript Ni kan jobba i grupp eller individuellt. Presentation av projekten sker nästa fredag 3 juni 9-12. Ni behöver inte skapa något power point presentation, det räcker att visa upp webbsiten och kod och prata om lösningen. Ni som känner att ni inte vill/kan redovisa för klassen, skicka hela lösningen inzippat till mig via chatten.

Uppgift

Skapa en webbsite som visualiserar förhållandet mellan olika observationer av en eller flera djurarter i något län i Sverige. Det kan t ex vara visualiseringen av älg och varg observationer eller rovfåglar och kaniner. Det är en grundidé som ni själva kan utveckla eller ändra. Ni kanske vill helt enkelt visualisera ned - eller uppgången av en viss art under vissa tidsperioder. Det går också att visualisera observationer i närheten av ett visst objekt, t ex någon fågelobservation i närheten av en flygplats. I så fall bör man skapa en statisk markör för flygplatsen. Sätt igång med projektet med ett brainstormingsmöte. Antingen gruppvis eller allihop. Tänk på vad som kan vara intressant att jobba med.

Tips

  • Återanvänd koden från övningarna
  • Ta små steg framåt, se till att design på sidan är tillfredställande och sedan börja
  • att fylla i den med innehållet. Inte tvärtom.

Krav

  • 🟡 Supabase med två eller flera tabeller
  • 🟡 PK/FK med de nya tabellerna
  • 🟡 SQL-satser testade på Servern
  • 🟡 Webbdesign enligt designprinciperna
  • 🔴 Websidan innehåller länkar till beskrivningar av de/den valda arten
  • 🟡 Data API så som Unshplash, Wikipedia mm.
  • 🔴 Skicka informationen om vädret under observationsdatumet
  • 🔴 Design av sidan med semantiska taggar
  • 🟡 CSS filen med alla element stylade. Minst en klass och id.
  • 🟡 Leaflet med markörer som representerar djurarten

Icke obligatoriska krav (om du har tid och lust att undersöka lite extra)

  • 🔴 Använd anropet till GBIF via dess API och sök direkt via ditt formulär.
  • 🟡 Manipulera kartbeteende med fler funktioner som vi inte gått igenom
  • 🔴 Möjliggör ifyllningen av en ny kommun i formuläret.
